# AI Agent 实战手册

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

企业级智能体开发完整指南 - 从理论到实践的系统性课程

## 课程简介

本课程深入探讨企业级AI Agent的设计与实现,核心聚焦于**代理性(Agenticness)**与**可控性(Controllability)**之间的平衡。通过理论讲解和实战项目,帮助开发者构建可靠、可控的智能体系统。

## 课程目录

1. **[可控性 vs. 代理性](./Course_Content/Chapter_01-Controllability_vs_Agenticness.md)** - 企业级Agent的核心矛盾
2. **[环境搭建与Hello Agent](./Course_Content/Chapter_02-Setup_and_Hello_Agent.md)** - 开发环境配置与第一个Agent
3. **[治理与控制](./Course_Content/Chapter_03-Governance_and_Control.md)** - Agent行为约束与权限管理
4. **[记忆与时间](./Course_Content/Chapter_04-Memory_and_Time.md)** - 短期与长期记忆设计
5. **[上下文工程](./Course_Content/Chapter_05-Context_Engineering.md)** - Prompt工程与上下文优化
6. **[自主行动层](./Course_Content/Chapter_06-Autonomous_Action_Layer.md)** - 工具调用与行动规划
7. **[编排与协作](./Course_Content/Chapter_07-Orchestration_and_Collaboration.md)** - 多Agent协同模式
8. **[中间件与路由](./Course_Content/Chapter_08-Middleware_and_Routing.md)** - 智能路由与中间件架构
9. **[AgentOps](./Course_Content/Chapter_09-AgentOps.md)** - 监控、日志与运维实践
10. **[SkillCraft框架设计](./Course_Content/Chapter_10-SkillCraft_Framework_Design.md)** - 技能抽象与框架设计
11. **[编排器Agent](./Course_Content/Chapter_11-Orchestrator_Agent.md)** - 任务协调与资源调度
12. **[代码库SkillKit](./Course_Content/Chapter_12-Codebase_SkillKit.md)** - 代码理解与生成能力
13. **[完整的AI代码助手](./Course_Content/Chapter_13-The_Complete_AI_Code_Assistant.md)** - 端到端项目实现

## 技术栈

- **语言**: Python 3.11+
- **核心框架**: LangChain, CrewAI, AutoGen
- **LLM**: OpenAI GPT-4, Claude, 其他兼容模型
- **开发环境**: Docker, VSCode Dev Containers
